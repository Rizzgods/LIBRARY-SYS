Welcome to the Library System

Here is a quick tutorial on how to use the system:

  1. Upload Books: Navigate to the Upload section to add new books to the library. You can fill in the book details and upload an image.
  2. View Books: Go to the Books section to see the list of available books. You can edit or delete books from here. Availability button can be toggled.
  3. Review Borrow Requests: Check the Borrow Requests section to approve or decline requests from users.
  4. Manage Recently Deleted Books: View and restore books from the Recently Deleted Books section.

For more detailed instructions, refer to the user manual or contact support.
